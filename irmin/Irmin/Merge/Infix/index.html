Module Irmin.Merge.Infix

Infix operators for manipulating merge results and promises.

open Irmin.Merge.Infix at the top of your file to use them.

Value Types

Merge Result Combinators

val (>>=*) : ('a, conflict) Result.result Lwt.t ‑> ('a ‑> ('b, conflict) Result.result Lwt.t) ‑> ('b, conflict) Result.result Lwt.t

>>=* is bind.

val (>|=*) : ('a, conflict) Result.result Lwt.t ‑> ('a ‑> 'b) ‑> ('b, conflict) Result.result Lwt.t

>|=* is map.

Promise Combinators

This is useful to manipulate lca results.

val (>>=?) : 'a promise ‑> ('a ‑> 'b promise) ‑> 'b promise

>>=? is bind_promise.

val (>|=?) : 'a promise ‑> ('a ‑> 'b) ‑> 'b promise

>|=? is map_promise.
