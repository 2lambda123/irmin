type spec
HTTP configuration.
val http_spec : ?strict:bool ->
       ?hooks:Irmin_http_server.hooks ->
       t -> spec
http_spec t returns the configuration for a server serving the contents of t. If strict is set, incoming connections will fail if they do not have the right X-IrminVersion headers.
