Module type Irmin.S_MAKER

module type S_MAKER = functor (S : Tc.S0) -> functor (C : Contents.S) -> functor (T : Tag.S) -> functor (H : Hash.S) -> S  with type step = S.t
       and type value = C.t
       and type tag = T.t
       and type head = H.t
S_MAKER is the signature exposed by any backend providing Irmin.S implementations. S is the type of steps (a key is list of steps), C is the implementation of user-defined contents, T is the implementation of store tags and H is the implementation of store heads.
Parameters:
S : Tc.S0
C : Contents.S
T : Tag.S
H : Hash.S
