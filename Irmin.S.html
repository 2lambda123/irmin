Irmin Store


type step
The type for step values.
include Irmin.Store.BC

An irmin store is a branch-consistent store where keys are lists of steps.

An example is a Git repository where keys are filenames, ie. list of '\'-separated strings. More complex examples are structured values, where steps might contains first-class fields accessors and array offsets.
