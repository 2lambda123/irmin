Module IrminNode

module IrminNode: sig .. end
Nodes represent structured values serialized in the block store.

The blocks form a labeled directed acyclic graph (DAG). For instance, using the following API it is possible to go from one node to another following a path in the graph. Every node of the graph might carry some optional contents.
