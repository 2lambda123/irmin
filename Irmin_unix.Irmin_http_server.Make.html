val listen : ?timeout:int ->
       ?strict:bool ->
       ?hooks:Irmin_unix.Irmin_http_server.hooks ->
       t -> Uri.t -> unit Lwt.t
listen t uri start a server serving the contents of t at the address uri. Close clients' connections after timeout seconds of inactivity. If strict is set (by default it is not), incoming connections will fail if they do not have the right X-IrminVersion headers.
