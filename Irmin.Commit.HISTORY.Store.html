val merge : t -> key Irmin.Merge.t
merge t lifts the merge functions defined over contents values to contents key. The merge functio will: (i) read the values associated with the given keys, (ii) use the merge function defined over values and (iii) write the resulting values into the store to get the resulting key.

If any of these operation fails, return `Conflict.
