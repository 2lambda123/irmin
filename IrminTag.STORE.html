Module type IrminTag.STORE

module type STORE = sig .. end

The *tag store* is a key / value store, where keys are names created by users (and/or global names created by convention) and values are keys from the block store.

A typical Irminsule application should have a very low number of keys in the tag store, are this store is not supposed to be really efficient.

include IrminStore.RW
module Key: IrminTag.S  with type t = key
Base functions over keys. 
module Value: IrminKey.S  with type t = value
Base functions over values.
