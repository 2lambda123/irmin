val config : ?config:Irmin.config ->
       ?root:string ->
       ?head:Git.Reference.t -> ?bare:bool -> ?level:int -> unit -> Irmin.config
Create a configuration value.

  • config is the optional initial configuration value. If not defined, it is Irmin.Private.Conf.empty
  • root is the local Git repository's root (the parent of the .git/ directory).
  • head is the name of the local Git repository's current branch. If set, this will cause the file root/.git/HEAD to be modified to contain ref: refs/heads/branch.
  • If bare is unset (the default), then the local Git repository's contents will be expanded into the filesystem on each update. This might cause some performance issues.
  • level is the Zlib compression level. If absent, use the default one.
