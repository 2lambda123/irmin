module Hash: Irmin.Hash.S
Hash provides base functions for contents hashes. 
val hash : Irmin.S.repo -> Irmin.S.contents -> Hash.t Lwt.t
hash r c it c's hash in the repository r.
val of_hash : Irmin.S.repo -> Hash.t -> Irmin.S.contents option Lwt.t
of_hash r h is the the contents object in r having h as hash, or None is no such contents object exists.
