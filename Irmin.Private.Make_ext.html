module Make_ext: 
functor (C : Contents.STORE) -> 
functor (N : Node.STORE  with type Val.contents = C.key) -> 
functor (H : Commit.STORE  with type Val.node = N.key) -> 
functor (T : Tag.STORE  with type value = H.key) -> 
functor (S : Sync.S  with type head = H.key and type tag = T.key) -> Irmin.S  with type step = N.Step.t
       and type value = C.value
       and type tag = T.key
       and type head = H.key
Make builds an high-level Irmin store using lower-level user-provided store and a native synchronisation implementation. See BC for details.
