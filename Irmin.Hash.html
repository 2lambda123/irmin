Module Irmin.Hash

module Hash: sig .. end
Hash provides function to digest user-defined serialized contents. Some backends might be parametrize by such a hash functions, other might work with a fixed one (for instance, the Git format use only SHA1).

An SHA1 implementation is available to pass to the backends.
