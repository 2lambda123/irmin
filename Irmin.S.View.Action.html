Module Irmin.S.View.Action

module Action: sig .. end

A view stores the list of Irmin.S.View.Action.tactions already performed on it. These actions are used when the view needs to be rebased: write operations are replayed while read results are checked against the original run.
type t = [ `List of key list * key list
       | `Read of key * value option
       | `Write of key * value option ]
Operations on view. The read results are kepts to be able to replay them on merge and to check for possible conflict: this happens if the result read is different from the one recorded.
