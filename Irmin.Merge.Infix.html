Module Irmin.Merge.Infix

module Infix: sig .. end
Infix operators for manipulating merge results and Irmin.Merge.promises.

open Irmin.Merge.Infix at the top of your file to use them.

Value Types



Merge Result Combinators


val ( >>=* ) : ('a, Irmin.Merge.conflict) Result.result Lwt.t ->
       ('a -> ('b, Irmin.Merge.conflict) Result.result Lwt.t) ->
       ('b, Irmin.Merge.conflict) Result.result Lwt.t
>>=* is Irmin.Merge.bind.
val ( >|=* ) : ('a, Irmin.Merge.conflict) Result.result Lwt.t ->
       ('a -> 'b) -> ('b, Irmin.Merge.conflict) Result.result Lwt.t
>|=* is Irmin.Merge.map.

Promise Combinators

This is useful to manipulate lca results.

val (>>=?) : 'a Irmin.Merge.promise ->
       ('a -> 'b Irmin.Merge.promise) -> 'b Irmin.Merge.promise
>>=? is Irmin.Merge.bind_promise.
val (>|=?) : 'a Irmin.Merge.promise -> ('a -> 'b) -> 'b Irmin.Merge.promise
>|=? is Irmin.Merge.map_promise.
