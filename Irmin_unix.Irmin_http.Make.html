Module Irmin_unix.Irmin_http.Make

module Make: Irmin.S_MAKER
High-level bindings. Most of the computation is done on the server, the client is (almost) stateless. The only thing that the client needs to remember is the tag of the current branch or the current head if the branch is detached.
