Functor Irmin_unix.Irmin_http.Low

module Low (M : Irmin.Metadata.S) : Irmin.S_MAKER
Low provides low-level bindings to the remote HTTP server.

Only the low-level operations are forwarded to the server, all the high-level logic is done on the client. Hence a high-level operation might take multiple RTTs.
