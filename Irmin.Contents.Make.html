The type for store paths. 
val merge : Path.t -> t -> key option Irmin.Merge.t
merge t lifts the merge functions defined on contents values to contents key. The merge function will: (i) read the values associated with the given keys, (ii) use the merge function defined on values and (iii) write the resulting values into the store to get the resulting key. See Irmin.Contents.S.merge.

If any of these operation fails, return `Conflict.
